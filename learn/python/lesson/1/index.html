Lesson 1

What's Python?

Python is a high-level, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small- and large-scale projects.


So what does that mean?


It means that Python is designed to be as close to colloquial English as possible. It has simple syntax (commands), punctuation (primarily colons), and blocks of code are defined by indentation, not curly braces. The language is meant to be very beginner friendly and that's [probably] why you're learning it. Another reason Python is so easy to learn is it's large community and plethora of documentation and tutorials.

Where do I start?

To start with Python, you first need to learn programming basics. These basics include knowledge of types, functions, classes, variables, and more. Once you get most of the basics down, you'll start with little pieces at a time and work your way up. It will take time and dedication to learn, so you have to be patient. You also want to make sure you test stuff you've learned, either by copying examples or creating projects. After lessons, I'll try to give you projects to test your knowledge.


Some websites you can use are w3schools, stackoverflow, and geeksforgeeks; but I'll try to cover as much as I can. Keep in mind that these websites are better to use as references than tutorials. It's usually best to google your question and check for trusted links like those I previously mentioned. In this lesson I'll try to teach native types, functions, and variables.

Types

Data types are used within type systems, which offer various ways of defining, implementing, and using them. Different type systems ensure varying degrees of type safety.

Almost all programming languages explicitly include the notion of data type, though different languages may use different terminology.

Common data types include integers, floating-point numbers, characters, strings, and booleans.

For example, in the Java programming language, the type int represents the set of 32-bit integers ranging in value from −2,147,483,648 to 2,147,483,647, as well as the operations that can be performed on integers, such as addition, subtraction, and multiplication. A color, on the other hand, might be represented by three bytes denoting the amounts each of red, green, and blue, and a string representing the color's name.

The basic types you need to worry about right now are integers, strings, and booleans. An int is essentially a number, a string is text that's not read by the code as syntax (strings are wrapped in quotation marks), and a boolean is a truthy or falsy value (True and False in python; case sensitive)

Examples:

some_string = "string"
some_int = 74
some_bool = True

Functions

Functions are a lot like algebraic functions, as in they take an argument (multiple arguments, or no arguments) and equate to a value (or, in this case, carry out tasks). A function is simply a “chunk” of code that you can use over and over again, rather than writing it out multiple times. Functions enable programmers to break down or decompose a problem into smaller chunks, each of which performs a particular task. Functions can be native (part of the programming language), user-defined, or accessed via API (application programming interface).

An example of a function is print. print is a native function, as in it comes with python pre-defined. The print function "prints" your text (string) of choice to the console (where you'll type in commands to run your code).

Because most functions have the ability to be re-defined (their "function" changes), typing a function's name alone will not do anything. To make a function "do it's thing", you must call it. In most programming languages, you do this by typing two parentheses after the name (just like algebra!). Inside these parentheses is also where you put arguments.

Example:

print("hi there") # printing "Hi there" to the console, notice that the argument is a string.

Variables

A variable is an abstract storage location paired with an associated symbolic name, which contains some known or unknown quantity of information referred to as a value; or in simpler terms, a variable is a container for a particular set of bits or type of data (like integer, float, string etc...). In Python, values can generally be accessed or changed at any time. Values are assigned to a variable with an equal sign (=). In Python, variables are automatically created when you type a name (that doesn't already exist).

Example:

my_message = "Sup, dude" # creating variable "my_message" and giving it a string value of "Sup, dude".
print(my_message) # printing the value of my_message ("Sup, dude") to the console, notice that the argument is NOT in quotation marks.

Downloading Python

Python is one of the easiest programming languages to download. If you're on Windows, it's as simple as the click of a button. If you're not on windows, downloading Python may take more effort. An alternative to downloading Python is using an in-browser IDE / REPL such as Replit

On windows, you'll have to open command prompt to use python. The command for running your python file is python my-file-name.py (python files have the py file extension).

MacOS works similar to Windows, however you'll need to open the terminal and not command prompt. Note that with older versions of Python, there is one macOS quirk that you need to be aware of: programs that talk to the Aqua window manager (in other words, anything that has a GUI) need to be run in a special way. Use pythonw instead of python to start such scripts.
